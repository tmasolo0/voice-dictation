# Voice Dictation

Голосовой ввод текста для Windows. Удерживай клавишу, говори, отпусти — текст появится в любом приложении.

Распознавание работает локально через [faster-whisper](https://github.com/SYSTRAN/faster-whisper) на NVIDIA GPU. Без интернета. Данные не покидают компьютер.

![Windows 10/11](https://img.shields.io/badge/Windows-10%2F11-blue) ![Python 3.11](https://img.shields.io/badge/Python-3.11-green) ![License MIT](https://img.shields.io/badge/License-MIT-yellow)

## Что умеет

- **Push-to-talk** — удерживай горячую клавишу, говори, отпусти — текст вставится в активное поле
- **Офлайн** — всё на твоём компьютере, без облака
- **Русский и английский** — автоопределение языка, можно переключаться между ними на лету
- **LLM-коррекция** — Qwen2.5 исправляет пунктуацию и термины (необязательно, нужна GPU с 12+ GB)
- **Словари терминов** — IT, математика, инженерия — подсказки для Whisper
- **Постобработка** — пунктуация, заглавные буквы, точка в конце
- **Управление моделями** — скачивание и выбор моделей Whisper прямо из интерфейса
- **Компактный виджет** — тонкая полоска на экране, цветом показывает состояние
- **Системный трей** — сворачивается, не мешает

## Два режима работы

### Без LLM (по умолчанию)

Whisper распознаёт речь, regex-процессоры приводят текст в порядок: пробелы, заглавные, точка в конце.

**VRAM:** ~3,5 GB (large-v3 в float16). Хватит любой NVIDIA GPU с 4+ GB.

### С LLM-коррекцией

После Whisper текст проходит через Qwen2.5-1.5B — она чинит пунктуацию, заглавные и ошибки распознавания. Regex-обработка работает **после** LLM, они дополняют друг друга.

**VRAM:** ~11 GB (Whisper ~3,5 GB + LLM ~7,5 GB). Для комфортной работы — GPU с 12+ GB (RTX 3060 12GB, RTX 4070 и выше).

> LLM работает в float32. На некоторых конфигурациях float16 даёт некорректный вывод при совместной работе с Whisper — приложение проверяет это автоматически и при проблемах переключается на float32. Без LLM приложение работает отлично.

## Требования

- **Windows 10/11**
- **NVIDIA GPU** с CUDA
  - Без LLM: 4+ GB VRAM
  - С LLM: 12+ GB VRAM
- **Microsoft Visual C++ Redistributable** (обычно уже стоит)

### VRAM по конфигурациям

| Конфигурация | VRAM | Пример GPU |
|-------------|------|-----------|
| Whisper large-v3 (float16) | ~3,5 GB | RTX 3060, RTX 4060 |
| Whisper large-v3-turbo (float16) | ~2,5 GB | GTX 1660, RTX 3060 |
| + LLM Qwen2.5-1.5B (float32) | ~11 GB | RTX 3080, RTX 4070 |
| Whisper medium (float16) | ~1,5 GB | GTX 1060 |

## Установка

### Инсталлятор (проще всего)

1. Скачай `VoiceDictation_Setup_1.0.8.exe` из [Releases](../../releases).
2. Запусти, следуй инструкциям.
3. При первом запуске модель Whisper large-v3 (~3 GB) скачается сама.

### Из исходников

#### 1. Python 3.11 (embeddable)

Скачай [Python 3.11 Windows embeddable (64-bit)](https://www.python.org/ftp/python/3.11.9/python-3.11.9-embed-amd64.zip) и распакуй в папку `python311/` внутри проекта.

Включи pip:
- Открой `python311/python311._pth` в блокноте.
- Раскомментируй строку `#import site` (убери `#`).
- Сохрани.

Установи pip:
```
cd python311
curl -O https://bootstrap.pypa.io/get-pip.py
python.exe get-pip.py
cd ..
```

#### 2. Зависимости

```
install_local.bat
```

#### 3. Запуск

| Файл | Описание |
|------|----------|
| `Dictation.vbs` | Без консоли (основной способ) |
| `Dictation.bat` | С консолью (для отладки) |

При первом запуске модель `large-v3` (~3 GB) скачается в `models/`.

#### Системный Python (альтернатива)

```bash
pip install PyQt6 pywin32 faster-whisper sounddevice numpy keyboard pyperclip pyautogui
python dictation.pyw
```

## Как пользоваться

1. Запусти Voice Dictation.
2. На экране появится виджет-полоска (зелёный — готов).
3. **Удерживай F9** — идёт запись (красный).
4. **Отпусти F9** — обработка (жёлтый) — текст вставляется в активное поле.
5. Виджет снова зелёный — можно диктовать дальше.

### Управление

| Действие | Как |
|----------|-----|
| Перетащить виджет | ЛКМ + перетащить |
| Контекстное меню | ПКМ на виджете |
| Настройки | ПКМ — «Настройки...» |
| Управление моделями | Настройки — вкладка «Модели» |
| Свернуть в трей | ПКМ — «Свернуть в трей» |
| Развернуть из трея | Двойной клик по иконке в трее |

## Настройки

Все настройки — через UI: **ПКМ на виджете — «Настройки...»**. Хранятся в `config.json` (создаётся при первом запуске).

### Основные

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Горячая клавиша записи | Клавиша для push-to-talk. Поддерживаются F-клавиши, буквы, комбинации (Ctrl+Space, Shift+Tab), отдельные модификаторы (Left Ctrl, Right Shift) | `F9` |
| Язык распознавания | «Авто» — определяет сам. Фиксированный язык ускоряет распознавание | Авто |
| Запускать при старте Windows | Автозапуск при входе в систему | Выкл |
| Запуск в свёрнутом виде | Сразу в трей, без виджета на экране | Выкл |
| Усиление микрофона | Коэффициент усиления. 1.0 — без усиления. Увеличь, если микрофон тихий | 1.0 |
| Звуковые эффекты | Сигнал при начале и окончании записи | Вкл |
| Приглушать звук при записи | Снижает громкость системных звуков, чтобы они не попали в микрофон | Вкл |

### Распознавание

#### Модель

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Модель | Модель Whisper. Требует перезапуск | `large-v3` |
| Устройство | CUDA (GPU) или CPU. GPU значительно быстрее. Требует перезапуск | CUDA (GPU) |
| Точность | float16 (быстро, меньше VRAM), int8_float16, int8 (минимум VRAM). Требует перезапуск | float16 |

#### Параметры качества

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Ширина поиска (beam) | Больше — точнее, но медленнее | 5 |
| Предыдущий сегмент как контекст | Для push-to-talk лучше выключить | Выкл |
| Порог сжатия | Фильтр повторов. Сегменты с высоким коэффициентом сжатия отбрасываются | 2.4 |
| Порог вероятности | Минимальная средняя log-вероятность. Неуверенные сегменты отбрасываются | −1.0 |
| Порог тишины | Вероятность «нет речи». Сегменты выше порога считаются тишиной | 0.6 |
| Штраф повтора | >1.0 снижает повторения | 1.2 |
| Размер N-грамм | Запрет повтора N-грамм подряд. 0 — без ограничений | 3 |
| Фильтр галлюцинаций | Длительность тишины (сек), после которой текст отбрасывается | 2.0 |

#### VAD (детектор речи)

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Чувствительность | 0.0–1.0. Меньше — чувствительнее | 0.50 |
| Мин. длина речи | Минимальная длительность речевого сегмента | 250 мс |
| Мин. длина паузы | Минимальная пауза для разделения сегментов | 500 мс |

### Постобработка

#### LLM-коррекция

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Использовать LLM | Включает коррекцию через Qwen2.5-1.5B. При первом включении предложит скачать модель (~3 GB). Нужно ~7,5 GB дополнительной VRAM | Выкл |
| Скачать модель | Кнопка появляется, если модель не установлена | — |

LLM исправляет пунктуацию (запятые, точки, тире), заглавные буквы (начало предложений, имена собственные) и очевидные ошибки распознавания.

#### Regex-обработка

Работает всегда (если включена), в том числе вместе с LLM.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Нормализация пробелов | Убирает лишние пробелы перед `.,:;!?` и добавляет после них | Вкл |
| Заглавная буква | Первая буква текста и буква после `.!?` — заглавная | Вкл |
| Точка в конце | Если текст не заканчивается на `.!?`, добавляет точку | Вкл |

### Словари

Доменные словари улучшают распознавание терминов через механизм hotwords.

| Словарь | Содержимое |
|---------|-----------|
| IT | Программирование, фреймворки, технологии |
| Math | Математические термины |
| Engineering | Инженерная терминология |

Можно включать несколько одновременно. Файл `dictionary.txt` в корне — базовый словарь, применяется всегда.

### Модели

Доступные модели Whisper — скачиваются и переключаются через UI.

| Модель | Размер | Качество | VRAM |
|--------|--------|----------|------|
| tiny | ~0,1 GB | Базовое | ~1 GB |
| base | ~0,2 GB | Низкое | ~1 GB |
| small | ~0,5 GB | Среднее | ~2 GB |
| medium | ~1,5 GB | Высокое | ~5 GB |
| large-v3-turbo | ~1,5 GB | Отличное | ~2,5 GB |
| **large-v3** | ~3 GB | **Лучшее** | ~3,5 GB |

**large-v3** — по умолчанию, лучшее качество. **large-v3-turbo** — быстрее при сопоставимом качестве, подойдёт при ограниченной VRAM.

## Словарь замен

Файл `replacements.json` — принудительные замены после распознавания (и после LLM, если включена):

```json
{
  "неправильное слово": "правильное слово",
  "питон": "Python",
  "гитхаб": "GitHub"
}
```

Замены регистронезависимые, работают по границам слов.

## Решение проблем

**Виджет не появляется.**
Проверь `logs/dictation.log`. Скорее всего, не хватает VRAM для модели.

**GPU не используется.**
Убедись, что стоят драйверы NVIDIA с CUDA. Или переключи устройство на CPU в настройках (будет медленнее).

**Плохое распознавание.**
- Добавь термины в `dictionary.txt` или включи доменные словари.
- Попробуй увеличить beam size (5 → 7).
- Включи LLM-коррекцию (если хватает VRAM).

**Не слышит микрофон.**
- Проверь, что микрофон выбран по умолчанию в настройках Windows.
- Увеличь усиление микрофона в настройках (> 1.0).

**Не хватает VRAM.**
- Возьми модель поменьше (large-v3-turbo или medium).
- Выключи LLM-коррекцию.
- Смени точность на int8 или int8_float16.

**LLM-коррекция не работает.**
- Проверь, что модель скачана (Настройки — Постобработка — «Скачать модель»).
- Нужно ~10 GB VRAM суммарно для Whisper + LLM.
- В логе видно, если приложение откатилось на float32 из-за проблем с float16.

**Текст не вставляется.**
- Voice Dictation вставляет текст через unicode SendInput — работает в большинстве приложений.
- Electron-приложения (VS Code, Slack) поддерживаются с v1.0.3.
