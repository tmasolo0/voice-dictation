# Voice Dictation

Голосовой ввод текста для Windows. Нажми клавишу — говори — текст вставляется в любое приложение.

Распознавание работает **полностью локально** через [faster-whisper](https://github.com/SYSTRAN/faster-whisper) на NVIDIA GPU. Интернет не нужен.

## Возможности

- **Push-to-talk** — удерживай F9, говори, отпусти — текст вставится в активное поле
- **Локальное распознавание** — всё офлайн, данные не уходят в сеть
- **Русский + английский** — автоопределение языка, смешанная речь
- **Режим перевода RU→EN** — F10 или ПКМ → "Перевод → EN"
- **Словарь терминов** — `dictionary.txt` с IT-терминами для точного распознавания
- **Постобработка текста** — автоматическая пунктуация, капитализация, точка в конце
- **Управление моделями** — скачивание и выбор моделей через UI (ПКМ → "Управление моделями...")
- **Минималистичный виджет** — круглый индикатор: зелёный = готов, красный = запись, жёлтый = обработка, синий = перевод
- **Системный трей** — сворачивается, прячется в fullscreen-приложениях

## Требования

- Windows 10/11
- NVIDIA GPU с CUDA (рекомендуется 6+ GB VRAM)
- ~1.5 GB для модели Whisper large-v3-turbo

## Быстрый старт

### 1. Скачай Python 3.11 (embeddable)

Скачай [Python 3.11 Windows embeddable (64-bit)](https://www.python.org/ftp/python/3.11.9/python-3.11.9-embed-amd64.zip) и распакуй в папку `python311/` внутри проекта.

Включи pip:
- Открой `python311/python311._pth` в блокноте
- Раскомментируй строку `#import site` (убери `#`)
- Сохрани

Установи pip:
```
cd python311
curl -O https://bootstrap.pypa.io/get-pip.py
python.exe get-pip.py
cd ..
```

### 2. Установи зависимости

```
install_local.bat
```

### 3. Запусти

| Файл | Описание |
|------|----------|
| `Dictation.vbs` | Запуск без консоли (основной способ) |
| `Dictation.bat` | Запуск с консолью (для отладки) |

При первом запуске модель `large-v3-turbo` (~1.5 GB) скачается автоматически в `models/`.

### Альтернатива: системный Python

```bash
pip install PyQt6 pywin32 faster-whisper sounddevice numpy keyboard pyperclip pyautogui
python dictation.pyw
```

## Использование

1. Запусти `Dictation.vbs`
2. На экране появится зелёный круглый виджет
3. **F9** (удерживай) — запись (виджет красный)
4. Отпусти **F9** — обработка (жёлтый) → текст вставляется в активное поле
5. **F10** — переключить режим перевода RU→EN (виджет синий + "EN")

### Управление

| Действие | Как |
|----------|-----|
| Перетащить виджет | ЛКМ + перетащить |
| Контекстное меню | ПКМ на виджете |
| Управление моделями | ПКМ → "Управление моделями..." |
| Переключить перевод | F10 или ПКМ → "Перевод → EN" |
| Свернуть в трей | ПКМ → "Свернуть в трей" |
| Развернуть из трея | Двойной клик по иконке |

## Модели Whisper

| Модель | Размер | Качество | VRAM |
|--------|--------|----------|------|
| **large-v3-turbo** | ~1.5 GB | Отличное | ~6 GB |
| large-v3 | ~3 GB | Лучшее | ~10 GB |
| medium | ~1.5 GB | Высокое | ~5 GB |

Модели можно скачать через UI (ПКМ → "Управление моделями...") или вручную с HuggingFace в `models/<имя>/`.

## Настройки

`config.json` создаётся автоматически при первом запуске. Основные параметры:

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `recognition.hotkey` | Горячая клавиша записи | `f9` |
| `recognition.model` | Модель Whisper | `large-v3-turbo` |
| `recognition.device` | Устройство | `cuda` |
| `recognition.beam_size` | Ширина beam search | `5` |
| `widget.size` | Размер виджета (px) | `150` |
| `widget.hide_in_fullscreen` | Скрывать в fullscreen | `true` |

### dictionary.txt

Словарь терминов — один на строку. Модель учитывает их при распознавании (hotwords API). Добавь специфичные слова: имена продуктов, технические термины, аббревиатуры.

## Решение проблем

**Виджет не появляется** — проверь `logs/dictation.log`

**GPU не используется** — убедись что установлен CUDA. Или смени `device` на `cpu` в `config.json`

**Плохое распознавание** — добавь термины в `dictionary.txt`

**Нет звука** — проверь что микрофон выбран по умолчанию в настройках Windows
