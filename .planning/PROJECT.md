# Voice Dictation Pro

## What This Is

Нативное Windows-приложение для голосового ввода текста с push-to-talk (F9). Локальное распознавание через faster-whisper на GPU (RTX 5090), минимум зависимостей от внешних API. Предназначено для личного использования и распространения — портативная папка с .exe, конфигом и моделями.

## Core Value

Мгновенная качественная диктовка с правильной пунктуацией и терминологией, работающая полностью локально на NVIDIA GPU.

## Requirements

### Validated

- ✓ Push-to-talk запись (F9) с визуальной индикацией состояния — existing
- ✓ Распознавание речи через faster-whisper (large-v3-turbo) на CUDA — existing
- ✓ Вставка текста в активное окно через clipboard — existing
- ✓ Конфигурация через config.json (модель, качество, горячие клавиши) — existing
- ✓ Поддержка нескольких моделей Whisper с переключением — existing
- ✓ VAD (Voice Activity Detection) для фильтрации тишины — existing
- ✓ Системный трей с контекстным меню — existing
- ✓ Пользовательский словарь терминов (dictionary.txt) — existing
- ✓ Базовый режим перевода (task="translate") — existing

### Out of Scope

- Облачные API для распознавания — противоречит принципу локальной работы
- Кроссплатформенность (Linux/macOS) — Windows-only приложение
- Многопользовательский режим — персональная утилита
- Формирование промптов/JSON/XML — отложено на будущие milestone
- Интеграция с LLM (Ollama/API) — отложено, исследуем позже
- Произвольные языковые пары перевода — в v1 только RU→EN

## Context

- Проект уже работает как Python-скрипт (dictation.pyw) с PyQt6 + faster-whisper
- Есть known issues: монолитный DictationWidget (682 строки), дублированная логика загрузки моделей, отсутствие тестов
- GPU: RTX 5090 с достаточным VRAM для large-v3 моделей
- Текущая модель: large-v3-turbo (float16, CUDA), beam_size=5
- Уже есть VAD, TTS-конфиг (xtts), секция OpenAI в конфиге (не используется)
- Пользователь диктует преимущественно на русском с вкраплениями английских IT-терминов

## Constraints

- **Platform**: Windows only — win32gui, win32api, NVIDIA CUDA
- **GPU**: NVIDIA с поддержкой CUDA (минимум 4GB VRAM для large-v3-turbo)
- **Privacy**: Минимум внешних API, максимум локальной обработки
- **Distribution**: Портативная папка — без установщика, без зависимостей на целевой машине
- **Stack direction**: Рассмотреть переход с Python на компилируемый язык (C++/Rust) для лёгкого .exe
