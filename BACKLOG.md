# Voice Dictation — Backlog

## v1.1 — Новый виджет

### Floating bar (стиль Aqua Voice)
- Заменить текущий круглый виджет на floating bar внизу экрана
- Дизайн на основе иконки приложения (Ava.jpg)
- Ждём скриншот Aqua Voice для референса

### Анимация при записи
- При нажатии Push-to-Talk: иконка в виджете анимируется (waveform / пульсация)
- При отпускании: анимация останавливается

### Audio ducking
- F9 down → приглушить системный звук (через pycaw / Windows Audio Session API)
- F9 up → вернуть громкость обратно
- Аналог поведения Windows при звонках (Teams/Skype)
